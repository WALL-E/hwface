# hwface
汉王人脸通考勤机Chrome插件

公司使用汉王人脸通考勤，无奈这个设备的UI只支持IE，chrome和firefox的兼容性都不好，居然不能使用日历插件选择日期，听说chrome插件专治这种病。


## 适用人群

* Mac用户
* Linux桌面用户
* IE怒用户


## 功能
* 一键查询最近7天的考勤记录

* 查询自定义考勤周期(Todo)

* 自动生成考勤异常报告(Todo)

## 安装
[点击下载](https://github.com/WALL-E/hwface/blob/master/dl/hwface-1.0.crx?raw=true)，直接拖入chrome浏览器即可完成安装
